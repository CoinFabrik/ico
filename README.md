# ISO Sample

## Requirements

- Node.js 8.1.2
- npm 4.6.1
- Truffle 3.2.2

## Commands

- `truffle compile`
- `truffle migrate`
- `truffle test`
- `npm run launch_testrpc`
- `npm run launch_ropsten`
- `npm run attach_ropsten`

In Truffle's commands you can specify the network to use using `--network`. Options are: `testrpc` and `ropsten`.
