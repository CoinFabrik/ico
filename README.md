# ISO Sample

## Requirements

- Node.js 8.1.2
- npm 4.6.1
- Truffle 3.2.2

## Commands

- `truffle compile`
- `truffle migrate`
- `npm run test
- `npm run ropsten`
- `npm run ropsten_attach`

In Truffle's commands you can specify the network to use with `--network`. Options are: `testrpc` and `ropsten`.
